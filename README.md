# Mini Book Crew

This project organizes a crew responsible for drafting small, self-contained books. The crew coordinates multiple writers and generates draft books based on a title and source text.

## Overview

- **MiniBookCrew** manages a list of writers.
- Writers can be added through the `add_writer` method.
- A draft book can be generated with `generate_book`, providing a title and the base text.

This simple structure serves as a foundation for future expansion, where more elaborate book creation workflows can be implemented.
